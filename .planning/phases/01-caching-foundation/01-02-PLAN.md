**tests/test_cache.py -- write these exact test functions:**

1. **`test_weights_cache_roundtrip()`** (NTBK-02):
   - Load trained_weights.json via json.load
   - Assert file exists and is not None
   - Assert "metadata" key exists
   - Assert "primary_ansatz" and "alternative_ansatz" keys exist
   - Assert primary_ansatz["weights"] is a list of 16 floats
   - Assert alternative_ansatz["weights"] is a list of 16 floats
   - Assert all weights are finite (not NaN, not Inf)
   - Assert primary_ansatz["name"] == "revised"
   - Assert alternative_ansatz["name"] == "first"

2. **`test_training_history_exists()`** (NTBK-02):
   - Load training_history.json
   - Assert "primary_ansatz" and "alternative_ansatz" keys exist
   - Assert each has "loss_values" as a non-empty list
   - Assert each has "batch_boundaries" as a list
   - Assert all loss values are finite positive floats
   - Assert loss_values list has at least 10 entries (COBYLA does many iterations)

3. **`test_hardware_cache_schema()`** (NTBK-03):
   - Load hardware_results.json
   - Assert "metadata" key exists with "backend_name", "is_live" keys
   - Assert "expectation_values" is a non-empty list of floats
   - Assert "predictions" is a non-empty list where each value is +1 or -1
   - Assert "test_accuracy" is a float between 0 and 100

4. **`test_cache_load_speed()`** (NTBK-04):
   - Import time module
   - Start timer
   - Load all three JSON files (trained_weights.json, training_history.json, hardware_results.json)
   - Assert total load time < 1.0 seconds (proxy for 30-second cell requirement -- JSON load is nearly instant)

5. **`test_dual_ansatz_cached()`** (NTBK-05):
   - Load trained_weights.json
   - Assert primary_ansatz["weights"] != alternative_ansatz["weights"] (different trained values)
   - Assert primary_ansatz["cnot_pairs"] != alternative_ansatz["cnot_pairs"] (different connectivity)
   - Assert len(primary_ansatz["cnot_pairs"]) == 6 (revised has 6 CNOT pairs)
   - Assert len(alternative_ansatz["cnot_pairs"]) == 3 (first has 3 CNOT pairs)

6. **`test_ansatz_cnot_difference()`** (NTBK-05 detail):
   - Load trained_weights.json
   - Assert alternative_ansatz["cnot_pairs"] == [[0,1],[1,2],[2,3]]
   - Assert primary_ansatz["cnot_pairs"] == [[0,1],[1,2],[2,3],[4,5],[5,6],[6,7]]

7. **`test_live_detection_priority(tmp_path, monkeypatch)`** (D-06 auto-detection):
   - Import load_hardware_results from cache_results
   - Create a temporary live_results.json in tmp_path with is_live=true and backend_name="test_live"
   - Create a temporary hardware_results.json in tmp_path with is_live=false and backend_name="test_cached"
   - Use monkeypatch.chdir(tmp_path) to temporarily change working directory
   - Call load_hardware_results() -- should return the live data
   - Assert result["metadata"]["is_live"] == True
   - Assert result["metadata"]["backend_name"] == "test_live"

8. **`test_json_schema_validation()`** (D-03 human-readable):
   - Load each JSON file as raw text
   - Assert each file contains newlines (pretty-printed, not minified)
   - Assert each file is valid JSON (json.loads succeeds)
   - Assert "metadata" key exists in all three files

All tests use plain `import json` and `pathlib.Path` for loading. Use `pytest.mark.skipif` if a cache file doesn't exist (allows running tests before cache generation for debugging).

Each test function has a clear docstring explaining which requirement it validates.
