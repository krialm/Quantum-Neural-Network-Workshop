4. **`generate_dataset(num_images)` function**: Copy EXACTLY from notebook cells 3-7. Must produce identical output with same random seed.    - ParameterVector("theta", length=2*num_qubits) -- NOTE: use "theta" not "θ" to avoid unicode issues in the standalone script
   - First layer: RY(params[i]) on qubit i for i in range(num_qubits)
   - CNOT layer: cx(pair[0], pair[1]) for each pair in cnot_pairs
   - Second layer: RX(params[num_qubits + i]) on qubit i for i in range(num_qubits)
   - Return the circuit

6. **`build_full_circuit(ansatz, num_qubits=8)` function**: Compose z_feature_map(num_qubits, parameter_prefix="a") with the ansatz into a single QuantumCircuit. The parameter_prefix="a" is critical -- it ensures feature map parameters sort alphabetically before ansatz parameters when Qiskit assigns them.
